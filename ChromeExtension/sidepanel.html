How it Works

ChristNote AI Trade combines computer vision with large language models to provide professional trading insights. It captures your screen, identifies chart patterns, indicators, and price action, then generates a detailed analysis based on your selected mode.

Setup Guide

1. Load Extension

Open chrome://extensions/, enable Developer mode, and load the extension folder.

2. Install Ollama (for local models)

Download from ollama.com and run a model (e.g., qwen2.5vl).

ollama run qwen2.5vl:7b

3. Ollama CORS Fix

Restart Ollama with OLLAMA_ORIGINS="chrome-extension://*" to allow connection.

OLLAMA_ORIGINS="chrome-extension://*" ollama serve

4. Cloud AI (Optional)

To use NVIDIA NIM text or vision models, go to ⚙️ Settings and enter your API key.

5. Voice Input

Click the 🎤 mic button to record. Supports Ukrainian, English, and 17 other languages. Set the language in ⚙️ Settings.
