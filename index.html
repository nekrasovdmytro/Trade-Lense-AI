Setup & Installation Instructions

Step 1: Install Chrome Extension

Download the Chrome Extension archive (.zip) above, unzip it, navigate to chrome://extensions, enable Developer Mode, and click Load Unpacked.

Step 2: Start Local Inference (Ollama)

Download and launch Ollama from ollama.com. Open your terminal and run ollama pull qwen2.5vl:3b (or your preferred text/vision model).

Step 3: Setup Cloud API (Optional)

For cloud models, create a free developer API key on build.nvidia.com, open the settings screen in the Chrome Extension sidepanel, and insert your NVIDIA key.

macOS "Malware / Unverified Developer" Warning

Because the macOS binary is compiled natively and not signed through the expensive Apple Developer Program, macOS Gatekeeper may block it. To run the app:

  • Option A: Right-click (or Control-click) AnalyzeScreen.app and select Open, then click Open Anyway.
  • Option B: Go to System Settings > Privacy & Security, scroll down to Security, and click Open Anyway next to the blocked warning.
  • Option C (Terminal): Run xattr -cr /Applications/AnalyzeScreen.app in your Terminal to clear the quarantine flag.
